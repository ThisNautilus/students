## 技术栈
- MVVM框架：vue.js(2.0)
- 脚手架：vue-cli
- 前端路由：Vue-Router
- 服务端通信：axios、json-server
- 第三方库：bootstrap
## 启动过程
- 1.先运行 `npm run json:server`来启动本地json接口。[本项目数据](https://github.com/ThisNautilus/jsonserver)
- 2.运行`npm install`安装依赖
- 3.运行`npm run serve`启动项目
